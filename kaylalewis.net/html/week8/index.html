Using jQuery

jQuery (whose motto is “write less, do more”) is a JavaScript library that avoids the tedious structure of JavaScript by condensing syntax and providing quick solutions to a multitude of JavaScript-related tasks. jQuery comes in handy for things like smooth scrolling and parallax effects, which is what we will cover today.

Plus this into the end of your webpage: 

			<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.2/jquery.min.js"></script>

Or, download jQuery and include the file in your site’s folder.

In today’s demo, we will implement smooth/responsive scroll + parallax effects into a basic page template. We will also take a look at Google Fonts.
